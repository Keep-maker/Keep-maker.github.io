# 梁道君 - 个人简历

这是一个基于 React + TypeScript + Vite 构建的个人学术简历网站。

## 技术栈

- **框架**: React 19
- **语言**: TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **图标**: Lucide React

## 功能特性

- 响应式设计，支持移动端和桌面端
- 平滑的动画效果和交互体验
- 学术论文展开/收起功能
- 一键导出 PDF
- 优化的性能和可访问性

## 部署

本项目已部署到 GitHub Pages，访问地址：https://keep-maker.github.io

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
